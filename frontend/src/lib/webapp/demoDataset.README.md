# Demo dataset

`demoDataset.js` — сгенерированный статический снимок для docs-demo и локального preview runtime.
Его нельзя править вручную: файл должен пересобираться целиком из заранее анонимизированного
снимка данных.

Текущий снимок описан в поле `DEMO_DATASET.generatedFrom`: источник — `tmp/postgres.dump`, созданный
28 мая 2026 года и уже очищенный от персональных данных до попадания в репозиторий. Сам dump не
коммитится.

## Как обновлять

1. Подготовить новый анонимизированный snapshot вне репозитория.
2. Пересобрать `frontend/src/lib/webapp/demoDataset.js` из этого snapshot тем же экспортёром, который
   формирует структуру `DEMO_DATASET`.
3. Проверить, что в `generatedFrom` обновлены источник, дата и счётчики.
4. Запустить `npm --prefix frontend run test` и `make check`.

Если понадобится ручная правка одного значения, сначала меняется exporter или исходный snapshot, а
затем заново генерируется весь файл.
